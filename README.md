# Mp3Cutter，个人原创的mp3剪切小工具。

> 该app使用了MD规范，界面风格简洁，功能上mp3剪切铃声制作，实用性比较强。 功能上虽然简洁，但是技术上该项目“麻雀虽小，五脏俱全”。
